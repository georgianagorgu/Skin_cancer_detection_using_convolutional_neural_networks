# Anexa - fragmente relevante din proiect
Acest fisier contine fragmentele importante din implementarea proiectului. Blocurile de cod sunt formatate cu text colorat si fundal transparent, astfel incat copierea din preview sa nu aduca fundal negru in Word.
## Fisiere importante
- `pipeline_rebuilt/config.py` - Centralizeaza caile pentru dataset, modele, rapoarte, grafice si pragurile de decizie folosite in evaluare si predictie.
- `pipeline_rebuilt/common.py` - Contine initializarea seed-urilor, generatoarele de imagini cu augmentare si metricele standard folosite la antrenare.
- `pipeline_rebuilt/train_efficientnet.py` - Construieste modelul pe baza EfficientNetB0 preantrenat, aplica doua etape de training si salveaza cel mai bun model dupa AUC pe validare.
- `pipeline_rebuilt/train_resnet.py` - Foloseste aceeasi strategie de transfer learning si fine-tuning pentru arhitectura ResNet50.
- `pipeline_rebuilt/model_registry.py` - Leaga fiecare model de fisierul salvat, functia de preprocesare, pragul de predictie si ultimul strat convolutional folosit pentru XAI.
- `pipeline_rebuilt/evaluate_model.py` - Cauta pragul optim dupa balanced accuracy, genereaza raport de clasificare si salveaza matricea de confuzie.
- `pipeline_rebuilt/predict_image.py` - Incarca modelul selectat, aplica preprocesarea corecta si scrie probabilitatile benign/malignant intr-un raport text.
- `pipeline_rebuilt/output_fusion.py` - Combina probabilitatile modelelor prin medie ponderata pe baza acuratetii pe train si produce raportul final al ansamblului.
- `pipeline_rebuilt/xai_methods.py` - Genereaza harti de activare pentru interpretarea deciziilor modelului si salveaza o figura comparativa cu imaginea originala.
- `pipeline_rebuilt/run_full_pipeline.py` - Orchestreaza pasii de antrenare, evaluare, comparare, output fusion, predictie si explicabilitate.
